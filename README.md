# AI Adaptive Training Coach

AI-система для генерации и адаптации тренировочной программы 
на основе профиля пользователя, истории тренировок и признаков 
восстановления.

В отличие от статических генераторов планов, система учитывает 
накопленный контекст: после каждой тренировки она анализирует 
выполненный объём, качество восстановления и признаки усталости — 
и возвращает структурированную рекомендацию по корректировке 
следующей сессии с объяснением принятых решений.

Реализовано как FastAPI-приложение с OpenAI-совместимым API 
и валидацией выхода по JSON-схеме.

---

## Что делает система

**Шаг 1 — Генерация стартового плана**  
Пользователь задаёт цель, уровень подготовки, доступное 
оборудование и ограничения. Система формирует персональный 
тренировочный план.

**Шаг 2 — Адаптация по истории**  
После каждой тренировки пользователь передаёт данные о сессии: 
упражнения, веса, подходы, субъективную сложность, качество 
восстановления. Система анализирует накопленную историю и 
предлагает корректировку следующей тренировки.

---

## Пример

Пользователь сообщает:
> «Жим лёжа 80 кг 3x5, последний подход не добил, сплю плохо 
> вторую неделю, завтра планировал присед»

Система возвращает:
```json
{
  "session_assessment": "признаки недовосстановления",
  "next_session": {
    "decision": "сохранить присед, снизить интенсивность на 10%, 
                 убрать дополнительный объём",
    "reasoning": "сочетание недосыпа, недовыполнения и накопленной 
                  усталости в последних сессиях"
  },
  "long_term": "оценить необходимость разгрузочной недели 
                в ближайшем микроцикле"
}
```

## Архитектура

| Компонент | Задача |
|---|---|
| Input Analyzer | Нормализует профиль и тренировочные записи, извлекает ключевые признаки |
| Context Builder | Подготавливает срез истории и RAG-контекст для принятия решения |
| Planner | Генерирует стартовый план или обновление существующего |
| Safety Agent | Проверяет результат на соответствие ограничениям и допустимым диапазонам нагрузки |
| Output Module | Приводит ответ к строгой JSON-схеме для валидации и тестирования |

## Ограничения

Система не предназначена для медицинской диагностики, лечения 
или рекомендаций при травмах. При наличии медицинского сценария 
в запросе система возвращает безопасный отказ без попытки 
строить тренировочную рекомендацию.
